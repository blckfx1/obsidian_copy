✅ Technical Document – Simple Obsidian-like App
Project Name: MiniVault (or whatever you like)
1. Project Overview
A minimal, fast, native Windows 10 desktop app that works with real Markdown files and folders (just like Obsidian).

Single .exe file
No Electron / HTML / web tech
Uses your existing folders as “vaults”
Minimal UI (very clean and lightweight)

2. Chosen Features (MVP)

Markdown editor with live preview
File explorer sidebar (folder tree)
Wiki-style [[internal links]] (click to open)
Tags / Search (simple full-text + tag search)
Daily Notes (one-click today’s note)
Works directly with real .md files and folders

5. How Core Features Work
Live Preview + Wiki Links

Use markdown library to convert to HTML
Render HTML in a Dear PyGui child window or use a simple custom renderer
Regex to detect [[Note Name]] → turn into clickable links

File Explorer

Use os.walk + Dear PyGui tree node
Double-click opens note

Daily Notes

Button → creates Daily Notes/2025-05-10.md automatically

Search

Simple full-text search across all .md files using whoosh or just pathlib + string search

Single Executable
Bashpyinstaller --onefile --windowed --icon=app.ico --name MiniVault main.py
